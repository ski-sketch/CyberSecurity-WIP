# Port Scanner

## Overview
A network reconnaissance tool to discover open ports and services on target systems.

## Features
- TCP/UDP port scanning
- Service version detection
- Multiple scan types (SYN, Connect, NULL, FIN, XMAS)
- Port range specification
- Multi-threaded scanning for performance
- Banner grabbing

## Learning Objectives
- Understanding network protocols (TCP/IP)
- Socket programming
- Network reconnaissance techniques
- Ethical hacking fundamentals

## Technologies
- Python socket library or Scapy
- Multi-threading/async operations
- Network packet manipulation

## Future Enhancements
- OS fingerprinting
- Vulnerability correlation
- Export results to various formats (JSON, XML, CSV)
- Stealth scanning techniques
- Integration with vulnerability databases

## Ethical Use
This tool is for educational purposes and authorized testing only. Always obtain proper authorization before scanning any network.
