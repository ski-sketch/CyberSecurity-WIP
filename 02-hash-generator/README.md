# Hash Generator

## Overview
A utility to generate and verify cryptographic hashes using various algorithms.

## Features
- Support for multiple hash algorithms (MD5, SHA-1, SHA-256, SHA-512)
- File and text hashing
- Hash comparison and verification
- Salt generation for password hashing
- HMAC generation

## Learning Objectives
- Understanding cryptographic hash functions
- Working with different hashing algorithms
- File integrity verification
- Security implications of different algorithms

## Technologies
- Python hashlib or Node.js crypto module
- Command-line interface
- File I/O operations

## Future Enhancements
- Bcrypt/Argon2 for password hashing
- Batch file processing
- Hash rainbow table detection
- GUI interface
